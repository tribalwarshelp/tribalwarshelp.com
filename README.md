## Development

### Prerequisites

1. Node.JS
2. yarn
3. gatsby-cli

### Installation

1. Clone this repo
```
git clone git@github.com:tribalwarshelp/tribalwarshelp.com.git
```
2. Navigate to the directory where you have cloned this repo.
3. ``yarn install``
4. ``yarn run develop``
